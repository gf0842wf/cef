# cef
cefpython3 based scriptable web browser for python.

# requirements

- python2.7
- wxpython
- cefpython3

# install

- install wxpython
```
osx:
  brew install wxpython
other:
  see wxpython official website
```    

- install cefpython3
```
see https://github.com/cztomczak/cefpython
```
- use cefjs

  copy the cefjs.py file to your project
